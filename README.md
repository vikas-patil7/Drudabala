# Drudhabala Website

The website app is in `artifacts/drudhabala`.

From the root folder you can run:

```bash
npm.cmd run dev
npm.cmd run build
npm.cmd run preview
```

In VS Code, use **Terminal > Run Task** and choose:

- `Drudhabala: dev server`
- `Drudhabala: build`
- `Drudhabala: preview`

You can also open `drudhabala.code-workspace` for a cleaner VS Code workspace view.
